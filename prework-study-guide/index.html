Prework Study Guide

HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The declaration tells the browser what document type is expected when opened
  • Declarations are not case-sensitive but HTML elements are case-sensitive
  • Metadata is webpage data that the browser uses to serve the webpage to users
  • The title element defines the title for a webpage; it is required and extremely important for Search Engine Optimization
  • The style element defines the CSS styles asociated with a website
  • The base element defines the base URL for a webpage
  • The link element connects an external resource to the HTML document
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport

CSS

  • CSS is a programming language used to define how webpage content is presented to users
  • CSS defines the colors, aesthetics, visual layout, and organization
  • CSS is used to define style rules for specific elements of an HTML page and format them
  • A margin property indicates how much space we want around the OUTSIDE of an element
  • The padding property adds space around the content INSIDE an element

Git

  • git status: checks which branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: will switch us to the main branch
  • git revert "SHA verion": will go back to version history and revert to expected/needed results
